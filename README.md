# yt-scene-renamer
Yeh ek standalone HTML utility tool hai jo browser mein directly khulta hai (koi install ya server nahi chahiye). Iska kaam hai un images/videos ko systematically rename karna jo AI tools (Gemini, Midjourney, Leonardo, Runway, etc.) se "Visual Scenes" aur "Video Motion" prompts ko copy-paste karke bahar generate kiye gaye hain
